# part3 实验记录：机器人 + 夹爪抓布接触验证

目标：验证 Franka 夹爪能在 Genesis（RDNA4）上对一块悬挂布料做**稳定接触抓取并抬起**，不发生粒子爆飞（PBD 接触是本项目最大不确定项）。
脚本：`scripts/30_franka_grasp_cloth.py`。

## 最终方案（跑通版）

竖直窗帘：`cloth.obj` scale=0.3，`pos=(0.40,0,0.55)`，`euler=(90,0,0)`，**固定顶边**（z 最高的 39 个粒子 `fix_particles`）→ 悬挂成帘。

抓取流程（分阶段，全部插值控制，避免高速扫掠）：
1. **静置** 250 步（臂在 home）。
2. **自标定夹爪朝向**：枚举 8³=512 个 euler 候选，在远离布的安全位 `set_dofs_position` 瞬移后**读两指 link 世界坐标**，算接近轴/手指分离轴；硬性要求接近轴≈+X（水平，`approach_x>0.85, |approach_z|<0.3`），再最大化手指轴沿 Y。
3. **抓近端(-X)竖边**：离机器人更近（在臂展内）、且水平接近时手掌停在布的 x 外侧 → 不与布面重叠。
4. 预抓取（布外侧，手指张开）→ 水平靠近把竖边送入两指之间 → 低力闭合（−4N）→ 抬起 0.18m。

| settle_only | 0.6257 | ✓ |
| after_calib | 0.6257 | ✓ |
| after_pre | 0.6257 | ✓ |

- 自标定结果：`euler=(45,90,45)`，`approach=[1,0,0]`（水平 +X）、`finger_sep=[0,1,0]`（沿 Y 骑跨布面）。
- 抓取区抬升 `grasp_region_dz=+0.17m`，`cloth_zmax 0.63→0.85`：**夹爪夹住竖边并把它提起**（顶边仍固定，被抓边升高→布料折叠变形）。
- 全程 `finite=True`、`cloth_zmax` 稳定，**无穿插无爆飞**。
- 渲染帧：`output/feature3/grasp/{01_safe_calib,04_close_*,05_lift_*}.png` 可视化确认。

## 踩坑与定位（关键）

1. **顶向下抓竖直布 → 爆炸**：手掌/手指与布面深度穿插。改为水平接近 + 抓边角，手掌停在布 x 范围外。
2. **布没固定 → 摊到地板**：悬挂窗帘必须 `fix_particles` 钉顶边。
3. **远端(+X)边够不到**：安全位距原点 ~0.97m > Franka ~0.85m 臂展，IK 全失败 → 自标定退化成垂直接近把布顶飞。改抓**近端(-X)边**（距原点 ~0.58m，可达）。
4. **决定性坑——自标定循环爆布**：早期对 512 个候选姿态逐个 `set_dofs_position` + `scene.step()`，很多 IK 解是手臂甩穿布料的怪姿态，每 step 一次就把布推爆（`after_settle cloth_zmax=18` 而非 0.63）。
   → **修复：标定阶段只 `set_dofs_position`（更新运动学）后直接读 link 坐标，绝不 `scene.step()`**，物理不推进 → 不碰布。这是本 feature 最重要的经验。

## 接触力度量说明

`get_links_net_contact_force()` 在抓取后读到 ~0：Genesis 1.1.1 中 **PBD 粒子与刚体的接触不计入刚体 net contact force**。因此接触有效性以「抓取区位移 / 抬升量」为判据，而非接触力读数。

## 结论

- ✅ RDNA4 上 Genesis 刚体夹爪 × PBD 布料接触抓取**可稳定跑通**，关键在控制流程（插值 + 低力 + 标定不 step），而非材质参数。
- 朝向不要靠肉眼/手算 quat，**自标定（瞬移读手指轴）**最稳。
- next-step（可选）：抓住后**解除顶边固定**再抬起，验证能否把整块布提离（当前是带固定顶边的折叠抬升）。
