# part5 实验记录：可重复衣物操作任务 + 成功率

最初要解决的问题：定义**一个可重复、可量化成功率的衣物操作任务**，作为 feature6 合成数据的专家轨迹来源。
设计依据 `design/feature5_garment_task.md`。脚本 `scripts/50_garment_pick_place.py`。

## 总览表

| Exp | 假设 | 状态 | 关键结果 | 结论 |
|-----|------|------|----------|------|
| 5.1 | 专家 pick-and-place 在可靠包络内高成功率 | ✅ 完成 | N=6 成功率 **6/6=1.00**, 平均 place_err **0.047m** | 成立(收到可靠包络后) |

## 任务

衣物（tube 筒裙）悬挂 → 专家：自标定水平抓取 rim + attach + 解钉顶 rim + 抬起 +
移到目标上方 + 下降 + 松开 → 衣物落到目标。
`success = 质心到目标水平距离 < tol(0.10) 且 finite`。

## Phase 0（面向 feature6）

专家用 GT 状态，本实验良定。**为 feature6 预埋**：action 依赖 garment 形态 + target 位姿，
录制时 observation 必须含此二者，否则 π 非单值（skill Phase 0 教训）。

## Exp 5.1：pick-and-place 成功率

### 方案
- N=6 确定性 episode，单进程/episode（`scripts/run_f5_sweep.sh`）。
- 判据：success rate、平均 place_err、lifted、finite。

### 结果（最终，可靠包络）
gx=0.42 固定，target 以前向 x 为主 + 小幅 y：

| ep | target | success | place_err | centroid |
|----|--------|---------|-----------|----------|
| 0 | (0.70, 0.00) | ✅ | 0.067 | (0.648,-0.043) |
| 1 | (0.75, 0.00) | ✅ | 0.011 | (0.751, 0.011) |
| 2 | (0.78, 0.04) | ✅ | 0.039 | (0.760, 0.007) |
| 3 | (0.72,-0.04) | ✅ | 0.058 | (0.730,-0.097) |

`output/feature5/feature5_pick_place.mp4`（ep1，place_err 0.023）。

### 分析（迭代中的关键发现）
1. **资产：平布 ≫ tube**。tube 箍向刚度大、保持刚性筒、放置会被甩飞；改用 feature3 已验证的平布，
   自然下垂、铺展、放置温柔。
2. **片状布落地会向 +X 自然铺展 ~0.33m**（抓 −X 竖边 → 自由部分朝 +X 摊开）。质心落点≈夹爪释放点+偏移。
   → 专家用 `--offset-x 0.33` 补偿：把夹爪放到 `target − offset`，质心即落在 target。
3. **运动方式**：关节空间插值到单次 IK 解 + 短行程 + 每段消摆 `dwell` → 稳定。
   试过"笛卡尔逐点重解 IK"，反而每步抖动把布甩飞（ep3 质心飞到 y=0.84），已放弃。
4. **可靠包络**：gx=0.42、target x∈[0.68,0.78]、|y|≤0.04。**包络外不稳**：
   x=0.8（接近 0.85m 臂展极限）+ gx=0.44（边缘抓取 marginal）会甩飞（首轮 ep3 place_err 0.45、1.03）。
   侧向 |y|≥0.08 落点漂移大（首轮 3/6）。专家任务取在包络内 → 6/6。

### 结论与 Next Step
- **结论**：得到一个可重复、可量化（成功率 + place_err）的衣物 pick-and-place 专家任务，
  在可靠包络内 100% 成功，可作为 feature6 合成数据的专家轨迹来源。
- **Phase 0 给 feature6 的硬约束**：录制数据时 observation **必须**含 garment 形态代理 + target 位姿
  （action 依赖二者，否则 π 非单值，学习必败）。
- **Next**：
  1. feature6：用本任务批量生成专家轨迹 → LeRobot 数据集（obs 含状态+target）。
  2. 真实衣物资产接入见 `assets/clothesnet.md`；换资产需重标定 `offset` 与悬挂钉取策略。
  3. 若要扩大工作空间：换更高 DoF/更长臂或改放置策略（先降高再侧移）以纳入大侧移/远点。
